# py09

このフォルダでは、宇宙テーマのデータ検証演習をします。
`Field` や `model_validator` を使って、入力に制約を付けたり、条件に応じて整合性をチェックしたりする方法を学びます。

## 何をする課題か

- `ex0/space_station.py`: `Field` を使って、宇宙ステーションのデータに制約を付けて検証する
- `ex1/alien_contact.py`: `model_validator` で、接触報告の条件付きルールを作る
- `ex2/space_crew.py`: ミッションやクルーの情報に条件を付けて、整合性を確認する

## ここで学ぶこと

- データに制約を付けて検証する
- 条件付きのバリデーションルールを作る
- 実際の入力データに対して整合性を確認する

## 実行方法

1. `cd py09/exN`
2. `python3 *.py`

`py09` は、より物語性のある演習を通じてロジック設計を学ぶステージです。
