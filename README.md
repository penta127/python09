# py09

宇宙テーマの演習フォルダです。
ステージ管理や状態管理、イベント処理に近い設計を学びます。

## どんなことをやるか

- `space_station.py`: 宇宙基地の状態管理とアクション処理
- `alien_contact.py`: 外部イベントの受け取りと応答ロジック
- `space_crew.py`: クルー情報の管理と役割分担

## 目的

- 状態遷移やイベント処理をプログラムで扱う力をつける
- データ構造とロジックの分離を意識する
- テーマのある設計をコードで表現する

## 実行方法

1. `cd py09/exN`
2. `python3 *.py`

`py09` は、より物語性のある演習を通じてロジック設計を学ぶステージです。
